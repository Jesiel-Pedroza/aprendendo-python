Estruturas Condicionais: Tomada de Decisão

As **estruturas condicionais** permitem que seu programa Python tome decisões. Elas avaliam uma condição e executam um bloco de código específico se essa condição for verdadeira (True), ou outro bloco de código se for falsa (False).

Pense nelas como um "caminho ramificado" no seu código: dependendo de uma situação, o programa segue por um caminho ou por outro.

1. A Declaração if (Se)

A declaração if é a forma mais básica de uma estrutura condicional. Ela executa um bloco de código APENAS se a condição especificada for verdadeira.

A sintaxe em Python é bem intuitiva:

Sintaxe Básica do if:

if condicao_a_ser_testada:
    # Este código será executado SE a condição for True
    # Note a INDENTAÇÃO (espaços ou tab) aqui! É CRUCIAL em Python.
print("Este código está fora do if e sempre será executado.")

ATENÇÃO! Indentação em Python:

Diferente de outras linguagens, Python usa a **indentação** (espaços ou tabulações no início da linha) para definir blocos de código. **Cada linha dentro de um if (ou else, elif, loops, funções) deve ter a mesma indentação.** Uma indentação incorreta causará um erro!

Exemplo de if:

idade = 20

if idade >= 18:
    print("Você é maior de idade.")
    print("Acesso permitido.")

print("Fim da verificação de idade.")

2. A Declaração else (Senão)

A declaração else é usada em conjunto com o if para definir um bloco de código que será executado **se a condição do if for falsa**.

Você usa else quando há duas possibilidades: uma se a condição é verdadeira, e outra se a condição é falsa.

Sintaxe do if-else:

if condicao:
    # Código executado se a condição for True
else:
    # Código executado se a condição for False

Exemplo de if-else:

idade = 16

if idade >= 18:
    print("Você é maior de idade.")
else:
    print("Você é menor de idade.")

3. A Declaração elif (Else if)

Quando você precisa testar **múltiplas condições em sequência**, usa-se o elif. Ele é uma combinação de else e if.

A primeira condição que for verdadeira será executada, e o restante das condições `elif` e o `else` serão ignorados.

Sintaxe do if-elif-else:

if primeira_condicao:
    # Código se a primeira condição for True
elif segunda_condicao: # 'else if'
    # Código se a segunda condição for True (e a primeira for False)
elif terceira_condicao:
    # Código se a terceira condição for True (e as anteriores forem False)
else:
    # Código se NENHUMA das condições anteriores for True

Exemplo de if-elif-else (Classificação de Notas):

nota = 8.5

if nota >= 9:
    print("Conceito A")
elif nota >= 7: # Se a nota não é >= 9, mas é >= 7
    print("Conceito B")
elif nota >= 5: # Se a nota não é >= 7, mas é >= 5
    print("Conceito C")
else: # Se a nota é menor que 5
    print("Conceito D")

Exercício 3.1: Classificação de Idade

Classifique uma idade em categorias: Criança, Adolescente ou Adulto.

  1. Crie uma variável idade = 15.
  2. Use if-elif-else:
    • Se idade for menor que 13, imprima "Criança".
    • Senão, se idade for menor que 18, imprima "Adolescente".

4. Combinando Condições com Operadores Lógicos

Você pode criar condições mais complexas combinando múltiplas expressões booleanas usando os operadores lógicos and, or e not que vimos anteriormente.

Exemplo de Lógicos com Condicionais:

idade = 22
possui_cnh = True
tem_restricao = False

if idade >= 18 and possui_cnh and not tem_restricao:
    print("Pode dirigir legalmente.")
else:
    print("Não pode dirigir ainda ou há impedimentos.")

Exercício 4.1: Condição para Concessão de Empréstimo

Um empréstimo é concedido se a renda mensal for maior que R$ 3000 E o histórico de crédito for bom.

  1. Crie variáveis renda_mensal = 3500 e bom_credito = True.
  2. Use um if com o operador and para verificar se o empréstimo pode ser concedido.
  3. Imprima "Empréstimo Aprovado!" ou "Empréstimo Negado.".
# Exercício 4.1
renda_mensal = 3500
bom_credito = True

if renda_mensal > 3000 and bom_credito:
    print("Empréstimo Aprovado!")
else:
    print("Empréstimo Negado.")

# Teste com renda_mensal = 2500, bom_credito = False

Um perfil especial é ativado se o usuário for administrador E não estiver bloqueado E sua idade for maior que 25.

  1. Crie variáveis: eh_admin = True, esta_bloqueado = False, idade_usuario = 28.
