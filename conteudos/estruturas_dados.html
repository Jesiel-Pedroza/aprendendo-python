Estruturas de Dados: Organizadores de Informação

Até agora, aprendemos a armazenar valores em variáveis individuais e a controlar o fluxo do nosso programa. No entanto, muitas vezes precisamos lidar com **múltiplos valores relacionados** ou grandes volumes de dados. É aqui que entram as **estruturas de dados** (ou coleções).

Estruturas de dados são maneiras eficientes de organizar, armazenar e manipular informações. Python oferece diversas estruturas de dados embutidas, cada uma com suas características e usos ideais.

Nesta lição, vamos explorar as coleções mais fundamentais em Python: **Listas, Tuplas, Dicionários e Conjuntos**.

1. Listas (list)

As **listas** são as coleções mais versáteis e usadas em Python. Elas são:

  • **Ordenadas:** Os itens mantêm a ordem em que foram inseridos.
  • **Mutáveis:** Você pode adicionar, remover ou modificar itens após a criação.
  • **Indexadas:** Cada item tem um índice numérico (começando em 0).
  • Podem conter itens de **diferentes tipos** de dados.

Listas são definidas usando **colchetes `[]`**.

Sintaxe Básica e Criação de Listas:

# Lista vazia
lista_vazia = []

# Lista de números inteiros
numeros = [1, 2, 3, 4, 5]

# Lista de strings
frutas = ["maçã", "banana", "laranja", "uva"]

# Lista mista (contém diferentes tipos de dados)
dados_pessoais = ["Alice", 30, True, 1.75]

print(f"Lista de números: {numeros}")
print(f"Lista de frutas: {frutas}")
print(f"Lista mista: {dados_pessoais}")

1.1. Acesso a Elementos (Indexação e Fatiamento)

Você pode acessar itens em uma lista usando seu índice (posição). O primeiro item tem índice 0.

Índices negativos começam a contar do final da lista (-1 é o último item).

**Fatiamento (Slicing)** permite pegar uma sub-lista.

Exemplos de Acesso e Fatiamento:

frutas = ["maçã", "banana", "laranja", "uva", "morango"]

# Acessando por índice
primeira_fruta = frutas[0]  # "maçã"
segunda_fruta = frutas[1]   # "banana"
ultima_fruta = frutas[-1]   # "morango"
penultima_fruta = frutas[-2] # "uva"

print(f"Primeira fruta: {primeira_fruta}")
print(f"Última fruta: {ultima_fruta}")

# Fatiamento (slice): [inicio:fim_exclusivo]
primeiras_duas = frutas[0:2] # ["maçã", "banana"]
do_segundo_em_diante = frutas[1:] # ["banana", "laranja", "uva", "morango"]
ate_o_terceiro = frutas[:3] # ["maçã", "banana", "laranja"]
copia_completa = frutas[:] # Cria uma cópia da lista

1.2. Modificando, Adicionando e Removendo Elementos

Como listas são mutáveis, você pode alterar seu conteúdo.

Exemplos de Modificação, Adição e Remoção:

minha_lista = ["pão", "leite", "ovos"]

# Modificar um elemento
minha_lista[0] = "café" # ["café", "leite", "ovos"]
print(f"Lista após modificação: {minha_lista}")

# Adicionar elementos:
# append(): adiciona no final
minha_lista.append("frutas") # ["café", "leite", "ovos", "frutas"]
print(f"Lista após append: {minha_lista}")

# insert(): adiciona em uma posição específica
minha_lista.insert(1, "queijo") # ["café", "queijo", "leite", "ovos", "frutas"]
print(f"Lista após insert: {minha_lista}")

# Remover elementos:
# del: remove pelo índice
del minha_lista[0] # Remove "café" -> ["queijo", "leite", "ovos", "frutas"]
print(f"Lista após del: {minha_lista}")

# remove(): remove pela primeira ocorrência do valor
minha_lista.remove("ovos") # ["queijo", "leite", "frutas"]
print(f"Lista após remove: {minha_lista}")

# pop(): remove e retorna o item pelo índice (ou o último se nenhum for dado)
item_removido = minha_lista.pop(0) # Remove "queijo" e o retorna
print(f"Item removido com pop: {item_removido}")
print(f"Lista após pop: {minha_lista}")

1.3. Outras Operações Úteis com Listas

Exemplos de Outras Operações:

numeros = [5, 1, 8, 2, 9]

# Tamanho da lista
tamanho = len(numeros) # 5
print(f"Tamanho da lista: {tamanho}")

# Verificar se um item existe na lista
existe_2 = 2 in numeros  # True
existe_7 = 7 in numeros  # False
print(f"O número 2 está na lista? {existe_2}")
print(f"O número 7 está na lista? {existe_7}")

# Ordenar a lista (altera a própria lista)
numeros.sort() # [1, 2, 5, 8, 9]
print(f"Lista ordenada: {numeros}")

# Reverter a ordem da lista (altera a própria lista)
numeros.reverse() # [9, 8, 5, 2, 1]
print(f"Lista invertida: {numeros}")

# Limpar todos os elementos da lista
minha_lista = ["a", "b", "c"]
minha_lista.clear() # []
print(f"Lista após clear: {minha_lista}")

1.4. Iterando sobre Listas com `for` (relembrando)

Loops for são perfeitos para percorrer todos os itens de uma lista.

Exemplo de Iteração de Lista:

disciplinas = ["Matemática", "Português", "História", "Ciências"]

print("Minhas disciplinas favoritas:")
for disciplina in disciplinas:
    print(f"- {disciplina}")

2. Tuplas (tuple)

As **tuplas** são muito parecidas com as listas, com uma diferença crucial: elas são **imutáveis**. Isso significa que, uma vez criadas, você não pode adicionar, remover ou modificar seus elementos.

Tuplas são definidas usando **parênteses `()`**.

São ideais para coleções de itens que devem permanecer constantes, como coordenadas de um ponto (x, y) ou informações fixas de configuração.

Sintaxe Básica e Criação de Tuplas:

# Tupla vazia
tupla_vazia = ()

# Tupla de coordenadas
coordenadas = (10.5, 20.3)

# Tupla de cores RGB
cor_vermelho = (255, 0, 0)

# Tupla mista
info_evento = ("Conferência Python", 2025, "São Paulo")

print(f"Coordenadas: {coordenadas}")
print(f"Info do Evento: {info_evento}")

2.1. Acesso a Elementos e Fatiamento (similar às Listas)

Assim como nas listas, você acessa elementos por índice e pode usar fatiamento.

Exemplos de Acesso e Fatiamento em Tuplas:

dias_semana = ("Segunda", "Terça", "Quarta", "Quinta", "Sexta", "Sábado", "Domingo")

primeiro_dia = dias_semana[0] # "Segunda"
ultimo_dia = dias_semana[-1]  # "Domingo"

print(f"Primeiro dia da semana: {primeiro_dia}")
print(f"Último dia da semana: {ultimo_dia}")

dias_uteis = dias_semana[0:5] # ("Segunda", "Terça", "Quarta", "Quinta", "Sexta")
print(f"Dias úteis: {dias_uteis}")

tamanho = len(dias_semana)
print(f"Número de dias na tupla: {tamanho}")

Diferença Chave: Imutabilidade

A principal diferença é que você **não pode** usar métodos como append(), insert(), remove() ou simplesmente atribuir um novo valor a um índice como minha_tupla[0] = "novo_valor" em tuplas. Isso resultará em um erro.

3. Dicionários (dict)

Os **dicionários** são coleções de pares **chave-valor**. Em vez de usar um índice numérico, você acessa os valores usando uma **chave** (que pode ser uma string, um número, ou uma tupla, desde que seja imutável).

Dicionários são:

  • **Não ordenados** (em versões mais antigas do Python), mas a partir do Python 3.7+ eles mantêm a ordem de inserção dos itens.
  • **Mutáveis:** Você pode adicionar, remover ou modificar pares chave-valor.
  • Cada **chave deve ser única**.

Dicionários são definidos usando **chaves `{}`**, com pares chave e valor separados por dois pontos (`:`).

Sintaxe Básica e Criação de Dicionários:

# Dicionário vazio
dicionario_vazio = {}

# Dicionário de informações de uma pessoa
pessoa = {
    "nome": "Mariana",
    "idade": 28,
    "cidade": "Rio de Janeiro",
    "email": "mariana@email.com"
}

# Dicionário de preços de produtos
precos = {
    "maçã": 2.50,
    "banana": 1.80,
    "laranja": 3.00
}

print(f"Informações da pessoa: {pessoa}")
print(f"Preços dos produtos: {precos}")

3.1. Acesso, Adição e Modificação de Itens

Exemplos de Operações com Dicionários:

aluno = {
    "nome": "Carlos",
    "matricula": "2023001",
    "curso": "Engenharia"
}

# Acessar um valor usando a chave
nome_aluno = aluno["nome"] # "Carlos"
print(f"Nome do aluno: {nome_aluno}")

# Modificar um valor existente
aluno["curso"] = "Ciência da Computação"
print(f"Curso atualizado: {aluno['curso']}")
print(f"Dicionário após modificação: {aluno}")

# Adicionar um novo par chave-valor
aluno["media"] = 8.5
print(f"Dicionário após adição: {aluno}")

# Remover um item
del aluno["matricula"] # Remove a chave "matricula" e seu valor
print(f"Dicionário após del: {aluno}")

# pop(): remove e retorna o valor para a chave especificada
curso_removido = aluno.pop("curso")
print(f"Curso removido com pop: {curso_removido}")
print(f"Dicionário após pop: {aluno}")

# Verificar se uma chave existe
if "nome" in aluno:
    print("A chave 'nome' existe no dicionário.")

if "endereco" not in aluno:
    print("A chave 'endereco' NÃO existe no dicionário.")

3.2. Métodos keys(), values() e items()

Esses métodos são muito úteis para obter partes específicas do dicionário ou para iterar sobre ele.

Exemplos de keys(), values(), items():

produto = {
    "nome": "Laptop Gamer",
    "marca": "TechPro",
    "preco": 5500.00,
    "estoque": 15
}

# Obter todas as chaves
chaves = produto.keys()
print(f"Chaves do dicionário: {list(chaves)}") # Convertendo para lista para visualização

# Obter todos os valores
valores = produto.values()
print(f"Valores do dicionário: {list(valores)}")

# Obter todos os itens (pares chave-valor como tuplas)
itens = produto.items()
print(f"Itens do dicionário: {list(itens)}")

3.3. Iterando sobre Dicionários

Você pode iterar sobre as chaves, valores ou pares chave-valor de um dicionário usando loops for.

Exemplo de Iteração de Dicionário:

usuario = {
    "username": "dev_python",
    "status": "online",
    "last_login": "2025-07-11"
}

print("Informações do Usuário:")
# Iterar sobre as chaves (padrão)
for chave in usuario:
    print(f"Chave: {chave}, Valor: {usuario[chave]}")

print("-" * 20)

# Iterar sobre os itens (pares chave-valor)
for chave, valor in usuario.items():
    print(f"{chave.capitalize()}: {valor}")

4. Conjuntos (set)

Os **conjuntos** são coleções **não ordenadas** de itens **únicos**. Eles são otimizados para operações de verificação de pertencimento e para remover duplicatas.

Conjuntos são definidos usando **chaves `{}`** (assim como dicionários, mas sem pares chave-valor) ou a função `set()`.

Sintaxe Básica e Criação de Conjuntos:

# Conjunto vazio (ATENÇÃO: {} cria um dicionário vazio. Use set() para um conjunto vazio)
conjunto_vazio = set()

# Conjunto de números (duplicatas são automaticamente removidas)
numeros_repetidos = {1, 2, 3, 2, 1, 4} # Resulta em {1, 2, 3, 4}
print(f"Conjunto de números: {numeros_repetidos}")

# Criar um conjunto a partir de uma lista
lista_com_duplicatas = ["a", "b", "c", "a", "d", "b"]
letras_unicas = set(lista_com_duplicatas) # {'a', 'b', 'c', 'd'}
print(f"Conjunto de letras únicas: {letras_unicas}")

4.1. Adicionar e Remover Elementos

Exemplos de Adição e Remoção em Conjuntos:

meu_conjunto = {"maçã", "banana", "uva"}

# Adicionar um elemento
meu_conjunto.add("laranja") # {"maçã", "banana", "uva", "laranja"}
meu_conjunto.add("maçã")    # Não adiciona, pois "maçã" já existe
print(f"Conjunto após adição: {meu_conjunto}")

# Remover um elemento
meu_conjunto.remove("banana") # {"maçã", "uva", "laranja"}
print(f"Conjunto após remoção: {meu_conjunto}")

# pop(): remove e retorna um item arbitrário (conjuntos não são ordenados)
item_removido = meu_conjunto.pop()
print(f"Item removido com pop: {item_removido}")
print(f"Conjunto após pop: {meu_conjunto}")

4.2. Operações de Conjuntos (Matemática)

Conjuntos suportam operações como união, interseção e diferença.

Exemplos de Operações de Conjuntos:

A = {1, 2, 3, 4}
B = {3, 4, 5, 6}

# União (elementos em A OU em B, sem duplicatas)
uniao = A.union(B) # {1, 2, 3, 4, 5, 6}
print(f"União de A e B: {uniao}")

# Interseção (elementos em A E em B)
intersecao = A.intersection(B) # {3, 4}
print(f"Interseção de A e B: {intersecao}")

# Diferença (elementos em A MAS NÃO em B)
diferenca = A.difference(B) # {1, 2}
print(f"Diferença de A - B: {diferenca}")

# Verificar subconjunto/superconjunto
C = {1, 2}
print(f"C é um subconjunto de A? {C.issubset(A)}") # True
print(f"A é um superconjunto de B? {A.issuperset(B)}") # False
