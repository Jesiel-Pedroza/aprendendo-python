Operadores em Python

Os **operadores** são símbolos especiais que realizam operações em valores e variáveis. Eles são a base para realizar cálculos, fazer comparações e tomar decisões em seus programas Python.

1. Operadores Aritméticos

Utilizados para realizar operações matemáticas comuns.

  • + : Adição
  • - : Subtração
  • * : Multiplicação
  • / : Divisão (retorna um float)
  • // : Divisão Inteira (retorna apenas a parte inteira)
  • % : Módulo (retorna o resto da divisão)
  • ** : Exponenciação (potência)

Exemplo de Operadores Aritméticos:

# Operadores Aritméticos
a = 15
b = 4

print("a + b =", a + b)   # Adição
print("a - b =", a - b)   # Subtração
print("a * b =", a * b)   # Multiplicação
print("a / b =", a / b)   # Divisão (resultado float)
print("a // b =", a // b) # Divisão inteira
print("a % b =", a % b)   # Módulo (resto da divisão)
print("a ** 2 =", a ** 2) # Exponenciação (a elevado a 2)

2. Operadores de Atribuição

Usados para atribuir valores a variáveis. Podem ser combinados com operadores aritméticos para uma sintaxe mais concisa.

  • = : Atribuição simples
  • += : Adicionar e atribuir (ex: x += 5 é o mesmo que x = x + 5)
  • -= : Subtrair e atribuir
  • *= : Multiplicar e atribuir
  • /= : Dividir e atribuir
  • //= : Divisão inteira e atribuir
  • %= : Módulo e atribuir
  • **= : Exponenciação e atribuir

Exemplo de Operadores de Atribuição:

# Operadores de Atribuição
saldo = 100

saldo += 50   # saldo = saldo + 50 (agora saldo é 150)
print("Saldo após adição:", saldo)

saldo -= 20   # saldo = saldo - 20 (agora saldo é 130)
print("Saldo após subtração:", saldo)

multiplicador = 2
multiplicador *= 3 # multiplicador = multiplicador * 3 (agora multiplicador é 6)
print("Multiplicador após multiplicação:", multiplicador)

3. Operadores de Comparação (Relacionais)

Usados para comparar dois valores. O resultado de uma comparação é sempre um valor booleano: True (verdadeiro) ou False (falso).

  • == : Igual a (atenção: um único = é atribuição, dois == é comparação!)
  • != : Diferente de
  • > : Maior que
  • < : Menor que
  • >= : Maior ou igual a
  • <= : Menor ou igual a

Exemplo de Operadores de Comparação:

# Operadores de Comparação
x = 10
y = 20

print("x == y ?", x == y)   # Falso
print("x != y ?", x != y)   # Verdadeiro
print("x > y ?", x > y)    # Falso
print("x < y ?", x < y)    # Verdadeiro
print("x >= 10 ?", x >= 10) # Verdadeiro
print("y <= 15 ?", y <= 15) # Falso

4. Operadores Lógicos

Usados para combinar expressões booleanas (True ou False).

  • and : Retorna True se AMBAS as condições são verdadeiras.
  • or : Retorna True se PELO MENOS UMA das condições é verdadeira.
  • not : Inverte o valor booleano (True vira False, False vira True).

Exemplo de Operadores Lógicos:

# Operadores Lógicos
tem_idade = True
tem_carteira = False

# Usando 'and'
pode_dirigir = tem_idade and tem_carteira # Falso, pois tem_carteira é False
print("Pode dirigir (idade E carteira)?", pode_dirigir)

# Usando 'or'
precisa_ajuda = not tem_idade or not tem_carteira # Verdadeiro (não tem carteira)
print("Precisa de ajuda (NÃO idade OU NÃO carteira)?", precisa_ajuda)

# Usando 'not'
nao_tem_idade = not tem_idade # Falso
print("Não tem idade?", nao_tem_idade)

Exercício 4.3: Condição Invertida

Verifique se uma pessoa **NÃO** está apta para um empréstimo. Ela não está apta se a renda for baixa OU se tiver dívidas.

  1. Crie variáveis renda_baixa = False e tem_dividas = True.
  2. Use o operador not combinado com or para verificar se a pessoa **não está apta**.
  3. Imprima o resultado.
# Exercício 4.3
renda_baixa = False
tem_dividas = True

# Condição para estar INAPTO: renda_baixa OU tem_dividas
esta_inapto = renda_baixa or tem_dividas

# Se a pessoa está INAPTA, então ela NÃO ESTÁ APTA
nao_esta_apta_emprestimo = esta_inapto # ou diretamente: (renda_baixa or tem_dividas)

print("Não está apta para o empréstimo?", nao_esta_apta_emprestimo)

renda_baixa = False
tem_dividas = False
nao_esta_apta_emprestimo = renda_baixa or tem_dividas
print("Não está apta para o empréstimo (ambos False)?", nao_esta_apta_emprestimo)
