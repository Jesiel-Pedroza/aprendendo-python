Entendendo Variáveis em Python

O que são Variáveis?

As variáveis são como "caixas" rotuladas onde podemos armazenar informações (dados) para usar depois em nosso programa. Cada caixa tem um nome (o rótulo) e guarda um valor (o que está dentro da caixa).

Em Python, criar uma variável e atribuir um valor a ela é muito simples. Você só precisa dar um nome à variável e usar o sinal de igual (=) para atribuir um valor.

Exemplo de Variável:

# Criando uma variável chamada 'nome' e atribuindo o texto "Alice" a ela
nome = "Alice"

# Criando uma variável chamada 'idade' e atribuindo o número 30 a ela
idade = 30

# Exibindo o conteúdo das variáveis
print(nome)
print(idade)

Lembre-se:

  • O sinal = é usado para atribuir um valor, não para comparar.
  • Nomes de variáveis devem ser descritivos, como nome_cliente, preco_total.
  • Em Python, você não precisa declarar o tipo da variável. Python detecta automaticamente!

Tipos de Dados Comuns

  • Strings (Texto): Ex: "Olá!", 'Python'
  • Inteiros: Ex: 10, -5, 1000
  • Floats: Ex: 3.14, 0.5, -1.2
  • Booleanos: True ou False

Exercícios de Variáveis

Abra o Google Colab e tente resolver os exercícios abaixo:

Exercício 1: Crie e Imprima suas informações

  1. cidade_natal como texto
  2. ano_nascimento como número
  3. Imprima uma frase com essas informações
# Exercício 1
cidade_natal = "Sua Cidade"
ano_nascimento = 1990

print("Minha cidade natal é", cidade_natal, "e nasci em", ano_nascimento)

Exercício 2: Troca de Valores

  1. Crie: a = 10 e b = 20
  2. Troque os valores
  3. Imprima os novos valores
# Exercício 2
a = 10
b = 20

# Troca com uma linha
a, b = b, a

print("Novo valor de a:", a)
print("Novo valor de b:", b)

Exercício 3: Cálculo Simples com Variáveis

  1. preco_produto = 45.50
  2. quantidade = 3
  3. total = preco_produto * quantidade
# Exercício 3
preco_produto = 45.50
quantidade = 3

total = preco_produto * quantidade

print("O valor total da compra é:", total)
