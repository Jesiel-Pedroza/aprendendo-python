Estruturas de Repetição (Loops): Automatizando Tarefas

Após aprender a tomar decisões com as estruturas condicionais (if, elif, else), o próximo passo fundamental na programação é entender como **repetir ações**. As **estruturas de repetição**, também conhecidas como **loops**, permitem que você execute um bloco de código várias vezes, de forma automática e eficiente.

Pense nos loops como uma forma de ensinar seu programa a realizar tarefas repetitivas sem a necessidade de escrever o mesmo código várias e várias vezes. Isso é a base da automação!

Em Python, os dois tipos principais de loops são o while (enquanto) e o for (para cada).

1. A Declaração while (Enquanto)

O loop while executa um bloco de código **enquanto uma condição específica for verdadeira (True)**. Ele continua repetindo até que a condição se torne falsa (False).

É ideal para situações em que você não sabe de antemão quantas vezes o loop precisará ser executado; a repetição depende de uma condição ser satisfeita ou não.

Sintaxe Básica do while:

while condicao_a_ser_testada:
    # Este código será executado REPETIDAMENTE ENQUANTO a condição for True
    # Lembre-se da INDENTAÇÃO! É CRUCIAL em Python.
    # Algo deve acontecer aqui dentro para que a 'condicao_a_ser_testada'
    # eventualmente se torne False e o loop termine.
print("Este código está fora do loop while e será executado após o seu término.")

ATENÇÃO! Cuidado com Loops Infinitos:

Se a condição de um loop while nunca se tornar False, o programa executará o bloco de código indefinidamente. Isso é chamado de **loop infinito** e fará com que seu programa trave. Certifique-se sempre de que, dentro do loop, algo altere a condição para que ela possa eventualmente se tornar falsa e o loop termine.

Exemplo de while: Contador Simples

# Exemplo: Contando de 1 a 5
contador = 1
while contador <= 5:
    print(f"Contador atual: {contador}")
    contador = contador + 1 # Incrementa o contador em 1 (pode usar 'contador += 1')

print("Contagem finalizada!")

Neste exemplo, o loop continua enquanto contador for menor ou igual a 5. A cada repetição, o valor de contador é incrementado, garantindo que a condição se tornará falsa quando contador atingir 6.

Exercício 1.1: Contagem Regressiva

Crie um programa que faça uma contagem regressiva de 10 até 1 e, ao final, imprima "Decolagem!".

  1. Use uma variável numero = 10.
  2. Use um loop while para continuar enquanto numero for maior ou igual a 1.
  3. Dentro do loop, imprima o valor de numero e depois decremente-o em 1.
# Exercício 1.1
numero = 10

while numero >= 1:
    print(numero)
    numero -= 1 # Equivalente a: numero = numero - 1

print("Decolagem!")

Exercício 1.2: Soma Acumulada

Crie um programa que calcule a soma dos números de 1 a 5.

  1. Crie uma variável num_atual = 1 e soma_total = 0.
  2. Use um loop while para continuar enquanto num_atual for menor ou igual a 5.
  3. Dentro do loop, adicione num_atual à soma_total e depois incremente num_atual.
  4. Ao final, imprima o valor de soma_total.
# Exercício 1.2
num_atual = 1
soma_total = 0

while num_atual <= 5:
    soma_total += num_atual # Adiciona o num_atual à soma_total
    num_atual += 1          # Passa para o próximo número

print(f"A soma dos números de 1 a 5 é: {soma_total}") # Deve ser 15

Exercício 1.3: Senha Simples (Loop com Entrada do Usuário)

Simule um sistema de login simples que pede uma senha até que a correta seja digitada.

  1. Defina uma senha_correta = "python123".
  2. Crie uma variável senha_digitada = "" (vazia inicialmente).
  3. Use um loop while que continue enquanto senha_digitada for diferente de senha_correta.
  4. Dentro do loop, use a função input() para pedir ao usuário que digite a senha.
  5. Quando a senha correta for digitada, imprima "Login bem-sucedido!"
# Exercício 1.3
senha_correta = "python123"
senha_digitada = "" # Inicialmente vazia para o loop começar

while senha_digitada != senha_correta:
    senha_digitada = input("Digite a senha: ") # Python espera entrada no console
    if senha_digitada != senha_correta:
        print("Senha incorreta. Tente novamente.")

print("Login bem-sucedido!")

Nota sobre input():

A função input() pausa a execução do programa e espera que o usuário digite algo e pressione Enter. Em um ambiente web (HTML/CSS/JS), essa interação seria feita de forma diferente, mas para o aprendizado de Python, imagine que você está executando este código em um terminal ou ambiente como o Replit.

2. A Declaração for (Para cada)

O loop for é usado para **iterar sobre sequências (como strings, listas, tuplas, etc.) ou outros objetos iteráveis**. Ele é frequentemente usado quando você sabe o número de vezes que o loop deve ser executado, ou quando você quer processar cada item em uma coleção.

Em Python, o for é muito versátil e intuitivo para percorrer coleções.

Sintaxe Básica do for com range():

for variavel_de_iteracao in range(valor_inicial, valor_final, passo):
    # Este código será executado para cada item/iteração no 'range'
    # 'variavel_de_iteracao' assume o valor de cada item em sequência.
    # INDENTAÇÃO aqui também!

Entendendo a Função range():

A função range() é muito comum com loops for para gerar sequências de números.

  • range(fim): Gera números de 0 até fim-1. (Ex: `range(5)` gera 0, 1, 2, 3, 4)
  • range(inicio, fim): Gera números de inicio até fim-1. (Ex: `range(2, 7)` gera 2, 3, 4, 5, 6)
  • range(inicio, fim, passo): Gera números de inicio até fim-1, pulando de acordo com o passo. (Ex: `range(0, 10, 2)` gera 0, 2, 4, 6, 8)

Exemplo de for com range():

# Exemplo: Iterando com range()
for i in range(3): # Gera 0, 1, 2
    print(f"Contando: {i}")

print("-" * 20) # Linha divisória

for ano in range(2020, 2024): # Gera 2020, 2021, 2022, 2023
    print(f"Estamos no ano: {ano}")

print("-" * 20)

for x in range(10, 0, -1): # Contagem regressiva de 10 a 1
    print(f"Contagem regressiva: {x}")

Exemplo de for Iterando sobre uma String:

# Exemplo: Percorrendo uma string
palavra = "Python"
for letra in palavra:
    print(f"Caractere: {letra}")

Exemplo de for Iterando sobre uma Lista (breve introdução):

# Exemplo: Percorrendo uma lista (veremos mais sobre listas em breve!)
frutas = ["Maçã", "Banana", "Cereja"]
for fruta in frutas:
    print(f"Eu gosto de {fruta}.")

O for é extremamente útil para trabalhar com coleções de dados, que serão exploradas em detalhes na próxima lição!

Exercício 2.1: Lista de Números Pares

Use um loop for para imprimir todos os números pares de 0 a 10 (inclusive).

  1. Utilize a função range() com o terceiro argumento para gerar apenas números pares.
# Exercício 2.1
for num_par in range(0, 11, 2): # Inicia em 0, vai até 10 (11-1), de 2 em 2
    print(num_par)

Exercício 2.2: Saudar Nomes

Dada uma lista de nomes, imprima uma saudação para cada um.

  1. Crie uma lista de strings: nomes = ["Alice", "Bob", "Charlie"].
  2. Use um loop for para iterar sobre esta lista.
  3. Para cada nome, imprima a mensagem: "Olá, [Nome]!".
# Exercício 2.2
nomes = ["Alice", "Bob", "Charlie"]

for nome in nomes:
    print(f"Olá, {nome}!")

Exercício 2.3: Tabuada

Gere a tabuada do 7 (de 7x1 até 7x10) usando um loop for.

  1. Use range() para gerar os multiplicadores de 1 a 10.
  2. Dentro do loop, calcule e imprima a multiplicação.
# Exercício 2.3
numero_tabuada = 7

for i in range(1, 11): # Multiplicadores de 1 a 10
    resultado = numero_tabuada * i
    print(f"{numero_tabuada} x {i} = {resultado}")

3. Comandos break e continue

Python oferece comandos especiais para controlar o fluxo de execução dentro de loops:

  • break: Interrompe o loop imediatamente. A execução do programa continua na primeira linha de código após o loop.
  • continue: Pula o restante do código da iteração atual do loop e avança para a próxima iteração.

Exemplo de break:

# Exemplo: Encontrando um número e parando o loop
numeros = [10, 20, 30, 40, 50]
alvo = 30

for num in numeros:
    if num == alvo:
        print(f"Número {alvo} encontrado! Parando a busca.")
        break # Interrompe o loop
    print(f"Verificando: {num}")

print("Loop finalizado (pelo break ou por terminar a lista).")

Exemplo de continue:

# Exemplo: Pulando um número em uma contagem
for i in range(1, 6):
    if i == 3:
        print(f"Pulando o número {i} nesta iteração.")
        continue # Pula para a próxima iteração do loop
    print(f"Processando número: {i}")

print("Contagem completa (com um pulo).")

Conclusão

As estruturas de repetição while e for são ferramentas incrivelmente poderosas para automatizar tarefas e processar grandes volumes de dados em Python. A prática com diferentes cenários é a chave para dominá-las.
