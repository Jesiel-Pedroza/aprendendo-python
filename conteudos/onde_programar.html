Vamos Programar em Python!

Onde Começar? Ambientes de Programação

Antes de tudo, precisamos de um lugar para escrever e executar nosso código Python. A forma mais fácil é usar um ambiente online que não exige nenhuma instalação no seu computador.

Minha sugestão é o Google Colab. Ele é gratuito, roda no navegador e já vem com Python configurado.

Como Acessar o Google Colab:

  • Abra seu navegador.
  • Acesse: colab.research.google.com
  • Faça login com sua conta Google.
  • Clique em "Novo notebook".
  • Pronto! Você pode digitar e executar código Python nas células.

Nosso Primeiro Código Python: "Olá, Mundo!"

Vamos escrever o clássico primeiro código:

Código Python:

print("Olá, Mundo!")

Como Executar no Google Colab:

  • Cole o código em uma célula do Colab.
  • Clique no botão "Play" ou pressione Shift + Enter.

Entendendo o código:

  • print() é uma função que exibe informações na tela.
  • Os parênteses () indicam uma chamada de função.
  • As aspas "" indicam uma string (texto).
